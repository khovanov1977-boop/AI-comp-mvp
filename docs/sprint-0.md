# Sprint 0 Scope

Sprint 0 implements the working technical skeleton from Project Blueprint v1:

- Next.js frontend
- FastAPI backend
- PostgreSQL persistence
- SQLAlchemy ORM
- Mock LLM, image, video, and voice providers
- Character creation
- Chat with mock AI replies
- Persisted message history

Out of scope:

- Real AI APIs
- Auth providers
- Payments
- Realtime voice
- Live avatars
- Mobile app
